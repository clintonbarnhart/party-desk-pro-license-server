# Version 4.0 Beta 1 — Signup Automation Studio

This release rebuilds Signup Automation into a professional customer-journey configuration experience.

## Included

- Fixed the compressed toggle layout shown in Phase 8.
- Added a responsive Automation Studio with workflow sections.
- Added live readiness cards for signup page, Square, plan mapping, and license delivery.
- Added Square plan mapping status with direct plan edit links.
- Added signup content, account access, subscription protection, and redirect settings.
- Added configurable failed-payment grace period and cancellation behavior.
- Added preview and signup-builder shortcuts.
- Preserved the automated Square checkout and license-provisioning workflow.

## Important

The Studio configures the existing self-service workflow. A paid plan still requires a valid Square subscription plan variation ID and working Square webhook configuration.
