# Version 4.0 Beta 2 — Square Plan Sync

This release removes the need to manually copy Square subscription plan variation IDs.

## Included

- One-click Square subscription plan synchronization
- Cached Square plan and variation catalog
- Human-readable plan labels with price and billing cadence
- Dropdown mapping in Signup Automation Studio
- Dropdown mapping in the individual Party Desk Pro plan editor
- Environment-aware Sandbox and Production caches
- Sync time and available-plan status
- Existing manual mappings remain preserved until changed

## Setup

1. Configure and test Square credentials.
2. Open **Party Desk Pro → Signup Automation**.
3. Click **Sync Square Plans**.
4. Select a matching Square subscription variation for every paid Party Desk Pro plan.
5. Save the Automation Studio.
