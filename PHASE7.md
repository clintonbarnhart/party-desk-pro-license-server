# Phase 7 — Commercial Platform Foundation

Version `3.7.0-alpha7` adds:

- A guided server setup wizard and readiness checks.
- Central platform settings for company, portal, support, update channel, and validation interval.
- A public product manifest endpoint without exposing package downloads.
- Expanded licensed update metadata including checksum, package size, plugin slug, release date, and support/portal URLs.
- An Update Integration admin page documenting all server endpoints and the client request contract.
- A kill switch for automatic update package delivery.

The signed release package URL remains available only after an active license and authorized website have been validated.
