# Phase 1 — Core Framework

Version: `3.1.0-alpha1`

## Delivered

- Stable plugin constants and service bootstrap
- Lightweight `PDP_` core autoloader
- Repeatable core migration runner
- Typed core settings facade
- Privacy-safe application logger with secret redaction
- Shared capability, nonce, URL normalization, and hashing helpers
- New **Party Desk Pro → System & Diagnostics** screen
- Safe database migration and log maintenance actions
- WordPress/PHP/database readiness reporting

## Compatibility

Phase 1 is additive and preserves the existing plans, requests, licenses, Square, subscription, portal, and email functionality.

## Verification

1. Activate or update the plugin.
2. Open **Party Desk Pro → System & Diagnostics**.
3. Confirm Database shows **Healthy**.
4. Save Core Settings.
5. Run Safe Migrations once and confirm a success notice.
